# Systems Biology Multimodel Inference

Bayesian model averaging across 10 competing ODE models of ERK/MAPK signaling, using PyMC SMC.

## Citation

Linden-Santangeli N, Zhang J, Kramer B, Rangamani P (2025). Increasing certainty in systems biology models using Bayesian multimodel inference. *Nature Communications* 16: 7307.
