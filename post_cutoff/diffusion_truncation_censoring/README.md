# Diffusion Model with Truncation and Censoring

Hierarchical drift-diffusion model handling censored and truncated reaction times in Stan.

## Citation

Henrich F, Klauer KC (2026). Modeling truncated and censored data with the diffusion model in Stan. *Behavior Research Methods* 58(2): 42.

## Links

- **Paper**: https://doi.org/10.3758/s13428-025-02822-z
- **OSF (code)**: https://osf.io/vg7zf/

## Metadata

- **PPL**: Stan (CmdStan >= 2.38 for `wiener_lcdf`/`wiener_lccdf`)
- **Language**: R
- **Published**: 2026
- **Scale**: 2 studies (56 and 116 subjects), 4 model variants, 8000 SBC fits
