# Hierarchical Supernova Dust Inference

GPU-scale hierarchical BayeSN model for supernova dust properties with physics-forward SED likelihood.

## Citation

Grayling M, Thorp S, Uzsoy AS, Narayan G, Mandel KS (2024). Scalable hierarchical BayeSN inference: Investigating dependence of SN Ia host galaxy dust properties on stellar mass and redshift. *MNRAS* (accepted). arXiv:2401.08755.
