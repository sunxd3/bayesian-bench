# Epidemic Modelling with Survey Covariates

Renewal-equation epidemic model with custom distributions in Turing.jl, comparing survey vs. mobility predictors.

## Citation

Koher A, Brinch CN, Hartvig H, et al. (2023). Epidemic modelling of monitoring public behavior using surveys during pandemic-induced lockdowns. *Communications Medicine* 3:116.

## Links

- **Paper**: https://doi.org/10.1038/s43856-023-00310-z
- **GitHub**: https://github.com/andreaskoher/Covid19Survey
- **Zenodo**: https://doi.org/10.5281/zenodo.7818793

## Metadata

- **PPL**: Turing.jl
- **Language**: Julia
- **Published**: 2023
- **Scale**: 5 Danish regions, 245 days, custom distributions (KLogistic, RandomWalk, NegBin2), LOO comparison
